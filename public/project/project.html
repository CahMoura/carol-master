BE THE HERO

Este projeto foi produzido na Semana Omnistack 11 da Rocketseat, com o objetivo de ajudar ONGs que podem se cadastrar pela aplicação web e pelo App. Após o cadastro é gerado um ID único para o cadastrado e outros podem fazer doações para ajudar as instituições. As tecnologias utilizadas foram Este projeto foi desenvolvido com as tecnologias abaixo: Node.js, Expo,Express, Axios, Cors, JavaScript, React Native e o Banco de Dados SQlite.


DevRadar

O DevRadar é um projeto desenvolvido durante a Omnistack 10, uma aplicação responsiva para Web e um App Mobile (tanto a parte Fontend como Backend) que visa conectar desenvolvedores próximos a você que trabalham com as mesmas tecnologias. Foi usado nesse projeto as seguintes tecnologias Node.js, React Native, Expo, Yarn, MongoDB e o Insomnia.
