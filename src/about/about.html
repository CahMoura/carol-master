DevRadar

Projeto DevRadar desenvolvido durante a Omnistack da @rocketseat . O DevRadar é um projeto que visa conectar desenvolvedores próximos a você que trabalham com as mesmas tecnologias. Desenvolvemos uma aplicação responsiva para Web e um App Mobile, tanto a parte Fontend como Backend, usando o Expo, Node.js, React, React Native, MongoDB, Yarn e o Insomnia.


Para ver o Projeto: DevRadar.


Be The Hero

Este projeto foi produzido na Semana Omnistack 11, com o objetivo de ajudar ONGs que podem se cadastrar pela aplicação web e pelo App. Após o cadastro é gerado um ID único para o cadastrado e outros podem fazer doações para ajudar as instituições. As tecnologias utilizadas neste projeto foram: Node.js, Expo, Express, ExpressJs, Nodemon, Axios, Cors, Eslint, Jest, JavaScript, React, React Native e para o Banco de Dados usamos o SQlite. Para ter acesso ao repositório: BE THE HERO.
